## 🔐 CertiVault Pro — Digital Certificate Verification System

A production-ready, client-side digital credential platform featuring blockchain-inspired 
verification, QR-code authentication, and comprehensive certificate lifecycle management. 
Built entirely with vanilla JavaScript and modern web technologies—no backend required.

### ✨ Key Features

- **🔏 Instant Certificate Verification** — Validate authenticity via unique Certificate ID 
  or QR code scan with simulated blockchain confirmation
- **📱 QR Code Integration** — Generate and scan QR codes for seamless mobile verification 
  using QRCode.js
- **📄 PDF Certificate Generation** — Export professionally designed certificates as PDFs 
  using jsPDF with custom branding
- **🎨 Live Certificate Preview** — Real-time visual preview during issuance with dynamic 
  avatar generation
- **📊 Analytics Dashboard** — Track total issued, verification counts, monthly statistics, 
  and blockchain blocks
- **⛓️ Blockchain Explorer** — Simulated distributed ledger with block hashes, validators, 
  and transaction history
- **🔒 Fraud Detection** — Automated invalid certificate handling with reporting mechanisms

### 🚀 Core Functionality

**Verification Flow:**
1. Enter Certificate ID (format: `CERT-2024-XXXXXXXX`) or upload QR code
2. System simulates blockchain lookup with loading animation
3. Displays verified recipient details, issuer info, and cryptographic hash
4. One-click PDF download and social sharing

**Issuance Flow:**
1. Fill recipient details (name, email, wallet address optional)
2. Configure certificate metadata (title, category, issuer, dates)
3. Preview certificate in real-time with dynamic styling
4. Generate unique ID, QR code, and blockchain hash
5. Auto-save to LocalStorage with explorer visibility
